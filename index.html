ProfitableCPMRate Documentation

Overview

ProfitableCPMRate is an automated script designed to simulate traffic and clicks, generating detailed metrics for a specific ad campaign. It tracks key performance indicators like Impressions, Clicks, CTR, CPM, and Revenue.

Features

  • Automated proxy rotation
  • Simulated clicks and traffic generation
  • Detailed performance metrics (CTR, CPM, Revenue)
  • Graphical and Markdown reports for performance tracking
  • GitHub Actions automation for periodic execution

Installation

To run the script locally, clone the repository and install the required dependencies:

      git clone https://github.com/Rekt-Developer/profitablecpmrate.git
      cd profitablecpmrate
      pip install requests matplotlib
      python profitablecpmrate_script.py

GitHub Actions Setup

This project can also be automated using GitHub Actions to run the script every 10 minutes. Check the .github/workflows/main.yml for more details on setting up the automation.

Customization

  • Change proxy URLs in the script.
  • Modify the revenue per click for better campaign results.
